Pequeño programa que limpia el formato de los datos .dat extraidos de trios para ser visualizados y proceados

1_start: es un pre formateo 
2_index_run: crea una matriz con los indices selecionados
3_plot_run: plotea las matrices creadas 

[pd: esto es un ejemplo de uso, por lo que para su correcto procesado se deberian cambiar pequeñas cosas en el codigo dependiendo de los datos]
